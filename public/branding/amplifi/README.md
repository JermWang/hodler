# AmpliFi Branding Assets

Place your branding files here:

- `logo-white.png` - White logo for dark backgrounds
- `logo-black.png` - Black logo for light backgrounds
- `logo-blue.png` - Primary blue logo
- `icon.png` - App icon / favicon source
- `icon.svg` - Vector icon
- `og-image.png` - Open Graph image (1200x630 recommended)
- `banner.png` - Banner image for social/promo

## Brand Colors

- **Primary Blue**: #1F4BFF
- **Deep Navy**: #0B1C3D
- **Accent Orange/Red**: #FF4D2D
- **Background Light**: #F7F8FA
- **Background Dark**: #000000

## Typography

- **Primary Font**: Plus Jakarta Sans
- **Fallback**: Inter
